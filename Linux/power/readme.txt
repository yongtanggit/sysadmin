Shell Scripts outputs power and batteries status from laptop. 

INTRODUCTION
------------

A laptop I am using doesn't have GUI installed. This shell scripts is written to output basic batteries status of a laptop.  

REQUIREMENTS
------------

1. This code was tested with Bash on a CentOS 7 host. 
2. Bc command is required. 

RUNNING THE PROGRAM
-------------------

No root privilege is required to run the program
